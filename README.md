# warmup-first-repo

Warmup repository for learning Git, GitHub workflows, version control, and basic release processes through hands-on practice.

---

## About

This repository is created as part of my warmup and onboarding tasks. It is used to practice and understand the core concepts of version control and collaborative software development using Git and GitHub.

---

## What This Repository Covers

- Git basics such as init, add, commit, push, pull, and merge  
- GitHub workflows and remote repositories  
- Branching strategies like main, dev, and feature branches  
- Understanding of basic release processes  
- Writing and maintaining proper documentation  

---

## Tools Used

- Git and GitHub  
- Visual Studio Code  
- Node.js and NPM  
- Postman / Thunder Client  
- Docker Desktop  

---

## Purpose

This repository is intended to:
- Practice Git and GitHub commands  
- Understand real-world development workflows  
- Track my learning progress  
- Serve as a reference for future projects
